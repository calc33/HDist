HCopyW <source-uri> <destination-dir> [options]
HCopyW copies files from <source-uri> listed in "_checksum.sha".

Options:
  --wait <file>
  -w <file>
      Wait for the <file> write lock to be released before starting copying.

  --run <executable>
  -r <executable>
      Starts the program specified by <executable> after copying is completed.

  --param <parameters>
  -p <parameters>
      Specify the parameters for executing the program specified by --run.
      If there are multiple parameters, enclose them in ".

  --compress <directory>
  -c <directory>
      Specify the directory that stores the compressed files for faster copy
      Available only if the source-uri is file://.
      If specified as a relative path, it will be a relative path from <source-uri>.

  --header <header>
      When <source-uri> is a http(s) URI, add a header to the HTTP request.

  --header-file <file>
      When <source-uri> is a http(s) URI, read headers from the specified file.

  --log <file>
  -l <file>
      If you want to output the log to a file, specify the file name.
      If you want to include the date and time in the file name,
      the following parameters will be replaced at run time
        %Y Year(4 digit)
        %M Month(2 digit)
        %D Day(2 digit)
        %H Hour(2 digit, 24 hours)
        %N Minute(2 digit)
        %S Second(2 digit)
        %P Process ID

  --help
  /?
      Show this help.
